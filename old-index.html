Footer

Footer Section 1

The whole footer is wrapped in a semantic footer tag; these can be used throughout the document, eg. footer information for a blog post etc. So it is suggested to add a class eg page-footer.

More Info

We haven't added the .container class to the footer, as we may want a full width BG color - which could be applied to the aforementioned page-footer class. Role=ContentInfo is applied to the primary footer as this contains supporting information for the whole page.
